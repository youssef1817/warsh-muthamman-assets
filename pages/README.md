# صور الصفحات

هذا المجلد يضم المرحلة النهائية الجاهزة من صور الطبعة:

- `warsh_muthamma_png/page001.png` إلى `warsh_muthamma_png/page485.png`
- `warsh_muthamma_png_index.json`: فهرس الصور النهائية بعد تخفيف الحجم

المراحل الوسطية مثل الاستخراج الخام والقص لا تحفظ هنا دائما، لأن السكربتات في `tools/` تستطيع إعادة إنتاجها عند الحاجة.

ترتيب العمل المعتمد:

1. استخراج الصفحات الخام من الـ PDF
2. قص الإطار والحواف مع الإبقاء على نفس أبعاد الصفحة
3. تخفيف حجم PNG
4. نشر مجلد `warsh_muthamma_png/` نفسه عبر GitHub Raw أو أي تخزين مباشر مماثل

لا توجد حزمة ZIP خاصة بالصفحات في هذا المسار الإنتاجي. التطبيق يحمّل الصفحات مباشرة ملفا ملفا.
