دليل الاختصارات

  • Shift + S: حفظ عام للصفحة الحالية، ويشمل بيانات الآيات وتخطيط الأسطر.
  • Ctrl + Z: التراجع عن آخر تعديل في الصفحة الحالية.
  • Ctrl + Y / Ctrl + Shift + Z: إرجاع آخر تعديل تم التراجع عنه.
  • Delete / Backspace: حذف المربع المحدد.
  • Shift + D: تكرار المربع المحدد (إنشاء نسخة مطابقة).
  • + / - : زيادة أو إنقاص رقم الآية.
  • Ctrl + أسهم يمين/يسار: تمديد حافة المربع لتلتصق بالهامش الأيمن/الأيسر.
  • Ctrl + مسافة (Space): تمديد المربع ليلتصق بالهامشين معاً.
  • تحديث أرقام اللاحقين: عند إنشاء مربع يزيد أرقام مربعات نفس السورة التي بعده، وعند الحذف ينقصها بواحد.
  • المحاذاة المغناطيسية: عند تفعيلها تلتقط حواف المربع للهوامش والدوائر والحدود القريبة. دبل كليك على مقبض اليسار/اليمين يمد الطرف للهامش الافتراضي.
  • الأسهم يمين/يسار: تقليب الصفحات (للسابق والتالي)
  • الأسهم أعلى/أسفل: الانتقال للمربع السابق/التالي
  • Shift + أسهم يمين/يسار: تحريك المربع أفقياً (يميناً ويساراً)
  • Ctrl + Shift + أعلى/أسفل: توسيع السطر (من الأعلى/الأسفل)
  • Ctrl + Shift + Alt + أعلى/أسفل: تضييق السطر (من الأعلى/الأسفل)
  • سحب وإفلات (بالفأرة): انقر على أي مربع واسحبه أفقياً أو عمودياً للعلامات.
  • Shift + سحب الدائرة: تحريك الدائرة أفقياً فقط (قفل التحريك العمودي).
  • Ctrl + Shift + سحب الدائرة: تحريك الدائرة عمودياً فقط (قفل التحريك الأفقي).
  • مع Ctrl + عجلة الماوس (Scroll): تكبير وتصغير الصورة.
  • مع الضغط المستمر على Alt + سحب: تحريك الصورة (Pan) في أي اتجاه.
  • انقر على النسبة % في شريط الزوم: العودة لحجم 100% الطبيعي.
