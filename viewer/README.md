# عارض ومحرر بيانات مصحف ورش المثمن

أداة ويب خفيفة ومستقلة (مبنية بـ HTML/CSS/JS و Node.js) صُممت خصيصاً لمطوري المشروع والباحثين. تهدف الأداة إلى معاينة صفحات "مصحف ورش المثمن" والتحقق البصري المباشر من صحة ودقة البيانات المستخرجة، **وتعديلها وحفظها بصرياً وبكل سهولة**.

## 🌟 المميزات الجديدة والقدرات

- **التعديل المباشر للمربعات (اللوحة اليمنى):** النقر على أي مربع آية أو علامة يفتح لوحة لتعديل إحداثيات العنصر (اليسار، اليمين، والمراكز). تُحفظ التغييرات تلقائياً في ملف `pages_json` التابع للصفحة.
- **الضبط الشامل للصفحة (اللوحة اليسرى):** التحكم بإزاحة جميع أسطر الصفحة لأعلى ولأسفل، وتغيير حجم الارتفاع (Scale). تنعكس التعديلات على كافة المربعات وتُحفظ فوراً في `page_layout_json`.
- **تصميم نقي وخفيف:** واجهة داكنة مريحة للعين تعرض الصفحة بأكملها بدون مشتتات.
- **التنقل السريع:** استخدام أسهم لوحة المفاتيح للتنقل بين الصفحات بسلاسة.
- **ذاكرة محلية:** يحفظ العارض آخر صفحة قمت بزيارتها وحالة التحديدات (مفعلة/معطلة).

---

## 🚀 كيفية التشغيل (تحديث هام)

من أجل دعم إمكانية **حفظ التعديلات في ملفات JSON مباشرة**، تم برمجة خادم (Server) مخصص لهذه الأداة باستخدام Node.js. (الخادم القديم `http-server` لم يكن يدعم الكتابة والحفظ).

لتعمل الأداة بكامل ميزاتها، افتح سطر الأوامر (Terminal) في المجلد الرئيسي للمشروع `warsh-muthamma-assets`، ونفذ الأمر التالي:

```bash
node viewer/server.js
```

### 🌐 تصفح المحرر
بمجرد تشغيل الخادم، يمكنك الوصول إلى الأداة عبر فتح الرابط التالي في المتصفح:
[http://localhost:8081/viewer/](http://localhost:8081/viewer/)
